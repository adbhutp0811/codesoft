# CodeSoft Web Development Internship Projects

A collection of front-end web development projects built during the **CodeSoft Internship**. Each project is a self-contained mini-application demonstrating core HTML, CSS, and JavaScript skills.

## Projects

| # | Project | Description |
|---|---------|-------------|
| 1 | [Calculator](/calculator) | A functional calculator supporting basic arithmetic operations |
| 2 | [Landing Page](/landing%20page) | A modern, responsive marketing-style landing page |
| 3 | [Portfolio](/portfolio) | A personal portfolio website showcasing skills and projects |

### 1. Calculator
A simple web-based calculator for performing addition, subtraction, multiplication, and division.
- **Files:** `calculater.html`, `style.css`, `script.js`
- **Features:** Basic arithmetic, clear/delete buttons, decimal input
- **[View README](/calculator/README.md)**

### 2. Landing Page
A responsive one-page website with a clean and professional design.
- **Files:** `index.html`, `style.css`
- **Sections:** Hero, About, Features, Services, Contact, Footer
- **[View README](/landing%20page/README.md)**

### 3. Portfolio
A personal portfolio website for **Adbhut Patel** with a downloadable resume and social links.
- **Files:** `index.html`, `styles.css`, `resume.pdf`
- **Sections:** About, Skills, Projects, Resume, Contact
- **[View README](/portfolio/README.md)**

Each project is a static website. To view any project:

1. Navigate to its folder
2. Open the main `.html` file in any modern web browser

Or use a live-server extension in VS Code for hot-reloading.
