# Calculator Project

This project is a simple web-based calculator built with HTML, CSS, and JavaScript.

## Features
- Basic arithmetic operations: addition, subtraction, multiplication, and division
- Clear (`C`) and delete (`DEL`) buttons
- Decimal input support
- Instant result calculation using the `=` button

## Files
- `calculater.html` — main calculator interface and logic

## How to Run
1. Open `calculater.html` in any web browser.
2. Use the buttons to perform calculations.

## Notes
- The calculator uses JavaScript's `eval()` for expression evaluation.
- For a production project, you may want to replace `eval()` with a safer parsing approach.
