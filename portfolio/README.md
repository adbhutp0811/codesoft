# Portfolio Website

This project is a personal portfolio website built with HTML and CSS. It showcases the owner's skills, projects, resume, and contact information in a clean and responsive layout.

## Features
- Responsive design for desktop and mobile devices
- Sections for About, Skills, Projects, Resume, and Contact
- Downloadable resume link
- Social links for GitHub and LinkedIn

## Project Structure
- `index.html` — main content and layout of the portfolio
- `styles.css` — styling and responsive design rules
- `*.png` and `*.jpg` files — images used in the portfolio
- `resume.pdf` — downloadable resume

## How to View the Project
1. Open `index.html` in your browser, or
2. Use a live server extension in your code editor to preview the site.

## Technologies Used
- HTML
- CSS
- JavaScript (for the year in the footer)
