# Landing Page Project

A simple and modern landing page built with HTML and CSS.

## Overview
This project is a responsive one-page website that includes:
- A hero section
- About section
- Features section
- Services section
- Contact section
- Footer

## Files
- `index.html` — Main structure of the landing page
- `style.css` — Styling and layout for the page

## How to Run
1. Open `index.html` in your browser, or
2. Use a live server extension in VS Code to preview the page

## Features
- Clean and professional design
- Responsive layout for different screen sizes
- Smooth section navigation with anchor links
- Attractive call-to-action button

## Project Purpose
This project is designed to showcase a basic landing page template that can be used for business websites, portfolios, or startup promotions.
